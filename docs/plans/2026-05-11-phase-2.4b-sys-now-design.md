## Architecture (one paragraph)

`:sys/now` is implemented as a thin "view" handler that nested-performs
`:clock/now` (the existing single-source-of-truth wall-clock op) and
returns the bare UTC-aware `dt.datetime`. Installing this as a separate
handler in `canonical_audit_stack` preserves the "only `clock.py` calls
`time.time()`" rule, gives replay-stability for free (the existing
`make_replay_clock_handler` continues to serve `:clock/now`
deterministically), and keeps `:sys/now` outside
`CANONICAL_AUDIT_WRAPPED_OPS` (mirrors `:clock/now`'s exclusion to
avoid recursion when audit-tier code reads time). Four call sites in
`http/routes/{blob,claim}.py` and `coder/{_session,_planner}.py`
migrate from `dt.datetime.now(dt.timezone.utc)` to
`substrate.effect.perform(":sys/now", {})` — all 1-line shape-
preserving swaps. The fifth wall-clock site (`_searcher.py:587`
`started_at_ms`) requires an integer-ms representation contract and
is W3-rescoped to a follow-up phase.

## Locked Decisions

### LD-1 — `:sys/now` handler shape: thin view over `:clock/now` (codex consensus Q1)

**Decision:** New standalone handler `make_sys_now_handler()` (factory in `src/persistence/effect/handlers/sys_now.py`) whose single clause body nested-performs `:clock/now`, converts the returned `{"ts": float}` to `dt.datetime.fromtimestamp(ts, tz=dt.timezone.utc)`, and returns the **bare** `dt.datetime` (no dict wrapper). Installed in `canonical_audit_stack` as a new handler positioned AFTER `clock` and BEFORE `audit` (so audit-middleware can call `:sys/now` if needed without recursion). NOT added to `CANONICAL_AUDIT_WRAPPED_OPS` (mirrors `:clock/now`'s exclusion).

**Codex consensus path:** Controller initial proposal was Option A (extend `make_system_clock_handler` with a second clause). Codex Round-1 verdict: **REJECT-FOR-NEW-OPTION-Z**. Three load-bearing arguments tipped:
1. G3 contract test (`tests/coder/test_steering_sys_now.py:108, 134`) assigns `sys_now_before = s.effect.perform(":sys/now", {})` directly and asserts `isinstance(..., dt.datetime)` — bare datetime, not `{"now": dt.datetime}`. Option A's "symmetry with `:clock/now`'s `{"ts": float}`" framing was a spec mismatch caught by reading the actual test.
2. Single source of truth for replay sequencing: Option Z keeps the "only `clock.py` calls `time.time()`" rule (codex audited `clock.py:1-5` docstring) and prevents three-factory parallel maintenance in the replay-clock family.
3. Audit-handler precedent: `audit.py:537-539` `mask(audit_name)` block documents "single primitive time source + view layers above it" as the intended architecture.

**Handler implementation sketch:**

```python
# src/persistence/effect/handlers/sys_now.py
import datetime as dt
from typing import Iterable
from persistence.effect.runtime import Handler, perform


def make_sys_now_handler(wraps: Iterable[str] = (":sys/now",)) -> Handler:
    """Return a handler that answers ``:sys/now`` with a UTC-aware datetime.

    Delegates to ``:clock/now`` (the single primitive wall-clock op) so
    that ``make_replay_clock_handler`` substitution propagates without
    needing a parallel ``:sys/now`` replay family. Returns BARE
    ``dt.datetime`` per the G3 contract.
    """
    def sys_now(args, k, ctx):
        ts = perform(":clock/now", {})["ts"]
        return dt.datetime.fromtimestamp(ts, tz=dt.timezone.utc)

    return Handler(
        name="sys_now",
        wraps=set(wraps),
        clauses={":sys/now": sys_now},
    )
```

### G2 — Replay-clock symmetry (LD-3 falsifiability for LD-1)

**Skeleton:**

def test_sys_now_delegates_to_replay_clock():
    from persistence.effect.handlers.clock import make_replay_clock_handler
    from persistence.effect.handlers.sys_now import make_sys_now_handler
    from persistence.effect.runtime import Runtime, perform
    import datetime as dt

    clock = make_replay_clock_handler([100.0, 200.0])
    sys_now = make_sys_now_handler()
    rt = Runtime(handlers=[clock, sys_now])

    with rt:
        t1 = perform(":sys/now", {})
        t2 = perform(":sys/now", {})

    assert t2 == dt.datetime.fromtimestamp(200.0, dt.timezone.utc)

### G3 — Wall-clock-ban scan over migrated sites (LD-4)

**Test:** `tests/test_wall_clock_ban_sys_now_sweep.py::test_no_wall_clock_in_migrated_provenance_sites`.

**R0-fold N1 widening:** scan matches `.now(` AND `.utcnow(` (regardless of attribute prefix `dt.datetime`, `datetime`, or alias). `_searcher.py` is exempted for `time.time_ns()` (site 6 deferred); entity-id sites using `uuid.uuid4().hex` remain (orthogonal noqa).

**Skeleton:**

```python
import ast
import pathlib

MIGRATED_FILES = [
    "src/persistence/http/routes/blob.py",
    "src/persistence/http/routes/claim.py",
    "src/persistence/coder/_session.py",
    "src/persistence/coder/_planner.py",
    "src/persistence/coder/_searcher.py",  # R0-fold I2 — site 5 added
]

BANNED_ATTRS = {"now", "utcnow"}


def test_no_wall_clock_in_migrated_provenance_sites():
    """LD-4 G3 — no datetime.now()/.utcnow() in the five migrated modules.

    Entity-id sites using uuid.uuid4().hex remain (orthogonal noqa).
    _searcher.py:587 (started_at_ms via time.time_ns()) is exempted
    until site 6 W3-rescope phase lands.
    """
    repo_root = pathlib.Path(__file__).parent.parent
    for relpath in MIGRATED_FILES:
        src = (repo_root / relpath).read_text()
        tree = ast.parse(src)
        for node in ast.walk(tree):
            if isinstance(node, ast.Call):
                func = node.func
                # Match X.now(...) / X.utcnow(...) — datetime call shapes
                if (isinstance(func, ast.Attribute)
                        and func.attr in BANNED_ATTRS):
                    # Allow only if the leftmost name is NOT datetime-like.
                    # In practice the migrated modules either use
                    # `dt.datetime.now()` or `datetime.now()`; both trip.
                    leftmost = func.value
                    while isinstance(leftmost, ast.Attribute):
                        leftmost = leftmost.value
                    leftmost_name = (
                        leftmost.id if isinstance(leftmost, ast.Name) else ""
                    )
                    if leftmost_name in {"dt", "datetime"}:
                        raise AssertionError(
                            f"{relpath}: {leftmost_name}.{func.attr}() call "
                            f"at line {node.lineno} — should route through "
                            f":sys/now"
                        )
```

**Falsifiability:** if any of the 5 migrations is forgotten or a future PR re-introduces a wall-clock read in these modules, the AST scan trips.
